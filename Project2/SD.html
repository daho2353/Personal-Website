Discord Bot

Private Discord bot programmed in Python that performs misc. tasks. It has been one of my favorite things to program and I update it whenever I have free time. The bot uses the Python Discord library and the asyncio library to monitor discord servers and respond to certain keywords. When making new features for the bot I usually try do something new programming wise, for example some functions use File I/O, some use Regex, if/elif trees, variable dictionaries, and many more. This project is probaly my most in depth project to date. Clicking on the image will link you to the Git Hub repository for this bot.
